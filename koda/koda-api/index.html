The KoDa API

About the KoDa API

What is the KoDa API ? The KoDa API provides access to the raw historical data collected from the following TrafikLab APIs: GTFS Regional Static data GTFS Regional Realtime NeTEx Regional Static data Who can access the dataset? Everyone who possess an API key can download (parts) the dataset. Apply for an API key. What period and public transportation authorities (PTAs) are included? The earliest data in the dataset is from February 2020 for the PTAs which data was available by the aforementioned TrafikLab APIs at that time.
