KoDa API

To access the KoDa data the user has to interact via the KoDa API. The KoDa API allows the user to access the raw GTFS data or make more advanced searches in the KoDa database via special API calls.

KoDa Database

The raw data has been downloaded from trafiklab.se and stored in its raw original GTFS format but the data has also been imported into a Cassandra database. FOr more information on data formats and how to use the KoDa Cassandra database please refer to the KoDa Database page
